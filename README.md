# FreeRTOS-Arduino
A [FreeRTOS](http://www.freertos.org/) port for the [Arduino Zero](https://www.arduino.cc/en/Main/ArduinoBoardZero).

## USAGE ##
Place the FreeRTOS folder to your arduino libraries directory.
